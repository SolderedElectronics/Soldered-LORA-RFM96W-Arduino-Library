static void setNextChannel ( uint16_t  start,
uint16_t  end,
uint16_t  count 
)
static

set LMIC.txChan to the next selected channel.

Parameters
[in]startfirst channel number
[in]endone past the last channel number

We set up a call to LMIC_findNextChannel using the channelShuffleMap and the channelEnableMap. We subset these based on start and end. start must be a multiple of 16.
